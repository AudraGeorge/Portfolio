My Approach

Every chaotic system has the same root problems. My work is to diagnose and solve them by focusing on five core areas of information management.

Taxonomy & Metadata

The fix isn't 'better folders,' it's metadata. I build the simple, documented blueprint that makes the system smart and the assets findable.

Info Architecture

When the structure makes no sense, work grinds to a halt. I tear down the maze of legacy folders and build a logical, scalable system with clear signs.

Workflow & Process

An "urgent" email is a workflow-killer. My fix is to build a door—a simple intake form or project hub process that converts vague asks into concrete, measurable tasks.

Governance & Standards

A system without rules will always fail. My fix is simple: I write things down. I build the clear, documented standards that make the system trustworthy and the work scalable.

Search & Discoverability

This is the payoff. A system is useless if you can't find things. When you build the core framework, you get discoverability. This is the proof that a clear system works.
